    # A is android App, S is server
    1.A->S:getversion
     .S->A:version number

    2.A-S:getdatabase
     .S-A:database, use Json
Stage2:
    3.A->S:send user/stime/etime/type
     .S->A:return available pid, use string

    4.A->S:send pid/user/stime/ctime/type
     .S->A:return auth_code(4 bits random numbers)
